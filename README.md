# Tour Guide – Tur Rehberi Web Sitesi

## Açıklama
Tour Guide, Türkiye’nin farklı bölgelerini tanıtan, kullanıcı dostu bir tur rehberi web sitesidir. Bu proje HTML, CSS ve JavaScript kullanılarak geliştirilmiştir ve içerisinde interaktif harita ve iletişim formu bulunmaktadır.

Site kullanıcılarına:
- Popüler tur bölgelerini (Doğu Anadolu, Karadeniz, Ege vb.) tanıtır,
- İletişim formu aracılığıyla mesaj göndermeyi sağlar,
- Google Maps API ile İstanbul’daki konumu gösterir.

## Özellikler
- Modern ve responsive tasarım
- Animasyonlu ve estetik butonlar
- Google Maps API ile interaktif harita
- İletişim formu (Ad, Email, Mesaj) – JS ile form doğrulama
- Footer ve sosyal medya ikonları

## Kullanılan Teknolojiler
- HTML5
- CSS3
- JavaScript 
- Google Maps API
